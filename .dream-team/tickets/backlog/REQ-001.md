# REQ-001: Kurly Impact Checker MVP 개발

## 연결 EPIC
EPIC-001

## 사용자 원문 요약
컬리 PM으로서 기획서 또는 텍스트로 "00기능 추가할건데 기획 및 개발 작업 영향도 파악해줘"라고 요청하면:
1. 해당 기능 개발/기획을 위한 작업 task 티켓 생성
2. 각 개발에 필요한 영향범위를 화면 및 기능단위로 체크
3. 어느 화면에 어떤 기능들에 개선/개발이 필요한지 파악
4. 백엔드/프론트엔드별 작업 사항 분류
5. 필요한 기획 및 확인해야 할 정책 분석
6. 난이도 점수화 (화면/기능 단위)
7. 플로우차트 형식으로 시각화 (시스템>화면>하위기능>모듈)
8. 종합 점수 및 영향도 등급 제안

## 추가 요구사항
- 프로젝트 코드 인덱싱으로 반복 분석 효율화
- Git 변경 감지 시 인덱스 자동 업데이트
- LLM 멀티 프로바이더 (3~4개, 사용자별 API 키)
- 기획서 입력: PDF, Confluence, 텍스트, 이미지 첨부
- 로컬 웹에서 결과 확인
- 티켓은 MD 형식 (추후 Jira 연동)
- 7명 동시 사용 가능
