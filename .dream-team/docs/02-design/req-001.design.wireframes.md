## 5. 인터랙션 시나리오

### 시나리오 1: 첫 분석 실행 (온보딩)

```
단계  사용자 행동                    시스템 반응                     화면
──── ───────────────────────────── ──────────────────────────────── ────
 1   /impact config 입력            LLM 설정 안내 표시               CLI
 2   프로바이더/키 입력              키 유효성 검증 -> 성공 메시지     CLI
 3   /impact init 경로 입력         프로젝트 감지 -> 인덱싱 시작      CLI
 4   (대기)                         프로그레스 바 -> 완료 요약         CLI
 5   /impact analyze 입력           기획서 입력 안내                  CLI
 6   기획 내용 텍스트 입력           파싱 결과 표시 + 확인 요청        CLI
 7   "Y" 입력 (확인)                분석 시작 -> 프로그레스 표시       CLI
 8   (대기)                         결과 요약 카드 표시               CLI
 9   "W" 입력 (웹 보기)             웹 서버 시작 -> 브라우저 열림      CLI+웹
10   대시보드 확인                   종합 요약 정보 표시               웹
11   "플로우차트" 탭 클릭            플로우차트 뷰 전환                웹
12   화면 노드 클릭                  하위 기능 노드 전개               웹
13   기능 노드 클릭                  하위 모듈 전개 + 상세 패널 표시   웹
14   상세 패널에서 점수 근거 확인    점수 산출 근거 상세 표시           웹
15   "체크리스트" 탭 클릭            기획 확인 사항 표시               웹
16   체크 항목 체크                  진행률 업데이트                   웹
17   "리포트 내보내기" 클릭          내보내기 모달 표시                 웹
18   "Markdown" 선택 + 내보내기     MD 파일 저장 -> 완료 알림          웹
```

### 시나리오 2: 반복 사용 (익숙한 사용자)

```
단계  사용자 행동                    시스템 반응                     화면
──── ───────────────────────────── ──────────────────────────────── ────
 1   /impact analyze 입력           인덱스 상태 확인                  CLI
 2   (자동)                         "변경 12건 감지, 업데이트 중..."  CLI
 3   기획 내용 입력                  파싱 + 확인                      CLI
 4   "Y" 입력                       분석 실행 -> 결과 요약            CLI
 5   "W" 입력                       웹 뷰 열림 -> 대시보드            웹
 6   플로우차트에서 특정 노드 확인   상세 패널 확인                    웹
 7   필터: "BE만" 선택               BE 관련 노드만 강조              웹
 8   티켓 탭에서 티켓 확인           티켓 카드 목록                    웹
```

 7   패널 닫기 (X)                  패널 닫힘, 노드 선택 해제         웹
 8   "FIT" 버튼 클릭                전체 보기로 복원                  웹
 9   "BE만" 필터 적용                BE 노드만 강조, 나머지 희미       웹
10   "전체" 필터 복원                모든 노드 정상 표시              웹
